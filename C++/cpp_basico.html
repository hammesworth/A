Sintaxe Básica e Estrutura

Todo programa C++ segue uma estrutura fundamental. O ponto de entrada de qualquer programa é a função main(). A linha using namespace std; nos permite usar nomes de elementos da biblioteca padrão (como cout) diretamente, sem o prefixo std::.

// Inclui a biblioteca de entrada e saída
#include <iostream>

// Permite usar nomes da biblioteca padrão diretamente
using namespace std;

// A função principal onde a execução do programa começa
int main() {
    // cout é usado para imprimir dados na tela
    cout << "Olá, Mundo!" << endl;

    // Retorna 0 para indicar que o programa terminou com sucesso
    return 0;
}
  • #include <iostream>: Inclui a biblioteca padrão de input/output (entrada e saída).
  • using namespace std;: Informa ao compilador que usaremos elementos do namespace (espaço de nomes) std. Isso evita ter que escrever std::cout ou std::endl.
  • int main(): Declara a função principal.
  • cout: É o fluxo de saída padrão (geralmente a tela). O operador << é usado para "enviar" dados para ele.
  • endl: Insere uma nova linha e limpa o buffer de saída.
  • return 0;: Indica ao sistema operacional que o programa foi concluído sem erros.

Variáveis e Tipos de Dados

Variáveis são contêineres para armazenar valores de dados. Em C++, você deve declarar o tipo da variável.

Tipos Primitivos Comuns:

  • int: para números inteiros (ex: 10, -5, 1000).
  • double ou float: para números de ponto flutuante (ex: 3.14, -0.5). double tem mais precisão.
  • char: para um único caractere (ex: 'a', 'B', '$').
  • bool: para valores booleanos, ou seja, true ou false.
  • string: para sequências de caracteres (texto). Requer a inclusão da biblioteca <string>.
#include <iostream>
#include <string> // Necessário para usar string

using namespace std;

int main() {
    int idade = 25;
    double altura = 1.75;
    char inicial = 'J';
    bool estudante = true;
    string nome = "João Silva";

    cout << "Nome: " << nome << endl;
    cout << "Idade: " << idade << endl;
    
    return 0;
}

Operadores Lógicos e Aritméticos

Operadores são usados para realizar operações em variáveis e valores.

Operadores Aritméticos

  • + (Adição)
  • - (Subtração)
  • * (Multiplicação)
  • / (Divisão)
  • % (Módulo - resto da divisão)

Operadores Lógicos

Usados principalmente em estruturas condicionais para combinar ou inverter condições.

  • && (E Lógico - AND): Retorna true se ambas as condições forem verdadeiras.
  • || (OU Lógico - OR): Retorna true se pelo menos uma das condições for verdadeira.
  • ! (Negação Lógica - NOT): Inverte o resultado de uma condição (true vira false e vice-versa).

Estruturas de Controle: if / else if / else

Essas estruturas permitem que o programa tome decisões e execute blocos de código diferentes com base em condições.

#include <iostream>

using namespace std;

int main() {
    int nota = 75;

    if (nota >= 90) {
        cout << "Conceito A" << endl;
    } else if (nota >= 70) {
        cout << "Conceito B" << endl; // Esta linha será executada
    } else if (nota >= 50) {
        cout << "Conceito C" << endl;
    } else {
        cout << "Reprovado" << endl;
    }

    return 0;
}

O programa verifica as condições em ordem. Assim que uma condição verdadeira é encontrada, seu bloco de código é executado e as demais condições (else if, else) são ignoradas.

Funções: Tipos, Parâmetros e Retorno

Funções são blocos de código que realizam uma tarefa específica e podem ser chamados (executados) de outras partes do programa. Isso ajuda a organizar e reutilizar o código.

Estrutura de uma Função

tipo_de_retorno nome_da_funcao(parametro1, parametro2, ...);

Tipo de Retorno

É o tipo de dado que a função "devolve" após sua execução. Se uma função não retorna nenhum valor, usamos o tipo void.

Parâmetros

São valores que a função recebe para trabalhar. Existem duas formas principais de passar parâmetros:

  • Passagem por Valor: Uma cópia do valor da variável é passada para a função. Alterações no parâmetro dentro da função não afetam a variável original.
  • Passagem por Referência (&): A função recebe uma referência (o endereço de memória) à variável original. Alterações no parâmetro dentro da função afetam a variável original.
#include <iostream>

using namespace std;

// Função que soma dois inteiros e retorna o resultado (retorno 'int')
int somar(int a, int b) { // 'a' e 'b' são parâmetros por valor
    return a + b;
}

// Função que não retorna valor (retorno 'void')
// e recebe um parâmetro por referência para modificá-lo
void dobrar(int& num) { // 'num' é um parâmetro por referência
    num = num * 2;
}

int main() {
    // Chamando a função 'somar'
    int resultado = somar(5, 3);
    cout << "A soma é: " << resultado << endl; // Saída: 8

    // Chamando a função 'dobrar'
    int meuNumero = 10;
    cout << "Número original: " << meuNumero << endl; // Saída: 10
    
    dobrar(meuNumero); // Passando a variável para ser modificada
    
    cout << "Número dobrado: " << meuNumero << endl; // Saída: 20

    return 0;
}
